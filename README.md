# ⚔ Diablo 3 — Season 39 Master Guide

A free, offline-friendly, installable web app covering **all 7 classes** for Diablo 3 **Season 39: Shades of the Nephalem** — build guides, a full Season Journey tracker, and a set of handy calculators. No login, no ads, no tracking. Your progress saves on your own device.

## 🔗 Live site
**https://ashervox.github.io/season-forge/**

## 📲 Install it (use it like an app, offline)
- **iPhone (Safari):** open the link → **Share** → **Add to Home Screen**.
- **Android (Chrome):** open the link → **⋮** menu → **Install app** / **Add to Home screen**.

You'll get an app icon that launches fullscreen and keeps working with no signal.

## ✨ Features
**Build Guide**
- Pick any of the **7 classes**, then choose a **goal build** (13 builds total; the free Haedrig's-Gift-set build is marked ★).
- Full endgame loadout: skills + runes, passives, key legendaries, Kanai's Cube powers, and legendary gems.
- A tailored, checkable **upgrade checklist** (knows the difference between free-set, farm-set, and Legacy-of-Dreams builds).
- **Leveling plan** and a **Paragon priority** guide per class/build.
- The whole UI **recolors to your class**, items are colored by rarity, and Priority steps give a satisfying ding + glow.

**Season Journey**
- Every tier (Chapters I–IV → Slayer → Champion → Destroyer → **Conqueror** → **Guardian**) with all objectives as checkboxes.
- All 10 Conquests, with the easy ones flagged.
- A live **season countdown**.

**Tools**
- **Kadala** blood-shard calculator
- **Cooldown Reduction** calculator (correct multiplicative stacking)
- **Augment** (Caldesann's) calculator — auto-matches your class's main stat
- **Gear gap-analysis** for your chosen build
- **Export / Import / Compare** progress codes — move between devices, or compare your Season Journey with a friend.

## 🔒 Privacy
Everything runs in your browser. Progress is stored locally (`localStorage`) on each device — nothing is uploaded. To sync or compare, use the export/import code in the Tools tab.

## 🛠 Tech
A single self-contained `index.html` (HTML/CSS/vanilla JS) plus a PWA `manifest.json`, a `sw.js` service worker for offline use, and icons. Hosted free on GitHub Pages.
